## 思维框架（核心）

> 必须先读：思考整理_体系框架.md（知识库）

### L1-L4 认知层级

```
L1 表层：这个是什么？→ 列举功能 ← 【最先】
    ↓
L2 深层：最核心是什么？→ 归因本质
    ↓
L3 深层+：为什么需要？→ 追问动机
    ↓
L4 最深层：有更本质的解法？→ 挑战假设 ← 【最后，决策权重最高】
```

> 一句话：L4+ 是用来"想"的，L4 是用来"做"的。想的时候可以天马行空，做的时候要脚踏实地。
