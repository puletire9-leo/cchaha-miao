## 思维框架（最高优先级）

### L1-L4 认知层级

```
L1 表层：这个是什么？→ 列举功能 ← 【最先，决策权重最低】
    ↓
L2 深层：最核心是什么？→ 归因本质
    ↓
L3 深层+：为什么需要？→ 追问动机
    ↓
L4 最深层：有更本质的解法？→ 挑战假设 ← 【最后，决策权重最高】
```

### 认知顺序 vs 决策优先级 vs 落地顺序

| | 认知顺序 | 决策权重 | 落地顺序 |
|---|---|---|---|
| L1 | 第1步（先做）| 最低 | 最后 |
| L2 | 第2步 | 中 | 倒数第二 |
| L3 | 第3步 | 高 | 正数第二 |
| L4 | 第4步（最后做）| **最高** | 第一 |

### 核心原则

- 认知顺序：先 L1 后 L4（怎么思考）
- 决策优先级：L4 权重最高（怎么决策）
- 落地顺序：L4→L3→L2→L1（反向实施）
- **一句话**：L4+ 是用来"想"的，L4 是用来"做"的。想的时候可以天马行空，做的时候要脚踏实地。
