## 思维框架（最高优先级）

### L1-L4 认知层级（深度思考框架）

> 来自：D:\知识库\memory\code\思考整理_体系框架.md

```
L1 表层：这个是什么？→ 列举功能 ← 【最先】
    ↓
L2 深层：最核心是什么？→ 归因本质
    ↓
L3 深层+：为什么需要？→ 追问动机
    ↓
L4 最深层：有更本质的解法？→ 挑战假设 ← 【最后】
```

### 任务复杂度决定深度

| 任务 | 需要层级 |
|------|----------|
| 简单（<1天）| L1 + L2 |
| 中等（1-3天）| L1 + L2 + L3 |
| 复杂（>3天）| L1 + L2 + L3 + L4 |

### 核心公式

```
思考过程：L1→L2→L3→L4（从已知到未知）
决策权重：L4>L3>L2>L1（深层优先）
落地顺序：L4→L3→L2→L1（反向实施）
```

### 关键原则

- **不能跳过**：L2依赖L1积累，L3依赖L2本质，L4依赖L3动机
- **L4+ 是用来"想"的，L4 是用来"做"的**
- **简单问题简单想，复杂问题复杂想**
- **认知顺序 vs 决策优先级 vs 落地顺序**
  - 认知顺序：先 L1 后 L4（怎么思考）
  - 决策优先级：L4 权重最高（怎么决策）
  - 落地顺序：L4→L3→L2→L1（反向实施）
